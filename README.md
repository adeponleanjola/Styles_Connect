# StyleConnect — Prototype Website
**Group 10 · SEN201 · Beauty & Grooming Platform**

---

## File Structure

```
styleconnect/
├── index.html   ← Main page structure (HTML only)
├── style.css    ← All styles, animations, and layout
├── app.js       ← All interactivity and data logic
└── README.md    ← This file
```

> **No images required** — all visuals use emoji characters and CSS gradients, so the site works offline with just these 3 files.

---

## How to Run

1. Keep all 3 files (`index.html`, `style.css`, `app.js`) in the **same folder**
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge)
3. No server or internet required (except for Google Fonts on first load)

---

## Features

| Feature | Description |
|---|---|
| Hero search | Type a service and press Enter or click "Find Now" |
| Service filter | Click any service card to filter the professionals list |
| Professional cards | Shows rating, distance, tags, pricing & availability |
| Map view | Mock map with interactive pins for each professional |
| Booking modal | Opens when any "Book" button or map pin is clicked |
| Toast notifications | Feedback messages for search, filter, and booking |

---

## Technologies Used

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, flexbox, animations
- **Vanilla JavaScript** — no frameworks or libraries
- **Google Fonts** — Playfair Display + DM Sans (loaded via CDN)
